# Рецепты

Публичное хранилище рецептов в формате Markdown для [Obsidian](https://obsidian.md/).

## Структура

- `Рецепты/` — рецепты по одному на файл;
- `Шаблоны/Рецепт.md` — шаблон новой карточки рецепта;
- `Вложения/` — изображения и другие файлы, используемые в заметках;
- `Добро пожаловать.md` — стартовая страница хранилища;
- `AGENTS.md` — правила для AI-агентов, работающих с репозиторием.

## Как добавить рецепт

1. Откройте хранилище в Obsidian.
2. Создайте заметку в папке `Рецепты`.
3. Вставьте шаблон `Рецепт` командой **Templates: Insert template**.
4. Заполните свойства, ингредиенты и шаги приготовления.
5. Добавляйте только те фотографии и источники, которые можно публиковать.

Имена файлов рекомендуется задавать по названию блюда, например `Тыквенный суп-пюре.md`.

## Синхронизация с GitHub

Remote `origin` уже настроен. После изменений их нужно зафиксировать коммитом и отправить в GitHub. Сам по себе подключённый remote не выполняет автоматическую синхронизацию; для неё можно использовать Git-клиент или плагин Obsidian Git.

## Публичность

Репозиторий публичный. Не добавляйте сюда пароли, токены, адреса, личные заметки и фотографии с конфиденциальными данными. Перед публикацией проверяйте права на тексты и изображения.

Лицензия пока не выбрана. Публичный доступ к репозиторию сам по себе не даёт разрешения копировать и распространять его содержимое.
